Un plugin de WordPress para ayudarte en el tedioso trabajo de tomar nota de los cambios en una web.

== Description ==

Plugin Pídeme Cambios es un Plugin para las Agencias Web con el que los clientes pueden crear notas rápidas de la web.

Este plugin puede ser muy interesante para los freelancers o personas que se dediques a programar y maquetar webs en WordPress. Porque gracias al plugin los clientes pueden utilizar las notas ancladas en la pantalla para perir cambios con frecuencia.

Permite guardar la información de forma ordenada, numerada, relacionada y enlazada.

== Installation ==

La instalacion de este plugin es igual a todos los de Wordpress.

e.g.

1. Subir el directorio `indianwebs-pideme-cambios` en la ruta `/wp-content/plugins/`
2. Activar el plugin en el apartado 'Plugins' del menú WordPress
3. Acceder al apartado Pideme Cambios en el menú de Wordpress y empieza a configurar el plugin.
